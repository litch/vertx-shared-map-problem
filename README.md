# Vert.x Shared Map Problem

NOTE: For use with Vert.x 3

## Instructions

Run the `run.sh` file.

## Contents

`main.rb`
Deploys the `put.rb` and `get.rb` verticles

`put.rb`
Writes an entry to a Vert.x shared map

`get.rb`
Reads the entry from the shared map

## Problem

The entry is not retrieved.
